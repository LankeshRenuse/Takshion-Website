# 🚀 Takshion Website

Modern futuristic website built with React, Vite, and Tailwind CSS.

## ✨ Features

* Responsive Design
* Smooth Animations
* Futuristic UI Effects
* Fast Performance

## 🛠️ Tech Stack

* React.js
* Vite
* framer-motion
* Tailwind CSS
* JavaScript

## ⚙️ Run Project

```bash
npm install
npm run dev
```

## 📦 Build

```bash
npm run build
```
